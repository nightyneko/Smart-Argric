# Smart Plant IoT System (Smart-Agric)

This project is an IoT-based environmental control system designed to maintain optimal conditions for plants. It monitors temperature, pressure, humidity, light intensity, and soil moisture, while controlling actuators like a fan, water pump, and OLED display. The system is integrated into a central dashboard via Node-RED using the MQTT protocol.

## System Components
- **STM32L432KC:** Main controller for sensors and actuators.
- **ESP32-S:** WiFi/MQTT bridge for remote communication.
- **Node-RED:** UI and backend logic.
